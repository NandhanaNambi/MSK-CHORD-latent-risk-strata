# My HDS 629 Project Page

A GitHub Pages site for my project.

## How to edit

1. Open `docs/index.html` in a text editor (or RStudio, or VS Code - whatever you use!).
2. Search for `CHANGE` to find every place you need to update.
3. Put your figures in `docs/figures/`.
4. Open `docs/style.css` to change colors or fonts.

## How to deploy

1. Put this repo on GitHub
2. Go to Settings → Pages
3. Set Branch: main, Folder: /docs
4. Your site will be live at: https://yourusername.github.io/your-repo-name

## File structure

    docs/
      index.html    ← the page (edit this)
      style.css     ← colors and fonts (edit this)
      figures/      ← put your figures here
        figure1.png
        figure2.png
